📲 How to Install & Keep Data Safe
📱 Install on Android (Chrome)
1. Open this HTML file in Chrome
2. Tap the ⋮ menu (top-right) → "Add to Home screen"
3. Tap Add — the app icon appears on your home screen
4. Open it like any other app — works offline too!
🍎 Install on iPhone (Safari)
1. Open this HTML file in Safari
2. Tap the Share button (box with arrow) at the bottom
3. Scroll down → tap "Add to Home Screen"
4. Tap Add — done!
⚠️ Getting a new/updated version of the app?
Your data lives in the browser's storage — it is NOT inside the HTML file.
So when you download a new version of the app:

Step 1: In the old app → go to ⚙️ Settings → tap ⬇ Export Backup
Step 2: Open the new app HTML file in Chrome
Step 3: Go to ⚙️ Settings → tap ⬆ Import Backup → pick the file from Step 1
Step 4: All your records, invoices & payments are restored ✅
💡 Best Practice
• Enable Auto-Save (tap 💾 in the top bar) → saves to a file automatically
• Or tap ⬇ Export Backup every evening and send the .json file to yourself on WhatsApp
• Keep the backup file in Google Drive so it is always safe
